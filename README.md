# Pomodoro Timer Like I Want It

Live: https://lakesare.github.io/time.

## Local Development

```bash
python3 -m http.server 8000
```

Open http://localhost:8000 and refresh (Ctrl+R) to see changes.

## Deploy to GitHub Pages

```bash
git add .
git commit -m "your message"
git push
```

Wait 1-2 minutes for GitHub Pages to rebuild.
